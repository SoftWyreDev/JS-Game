# 2D Side-Scrolling JavaScript Game

A 2D desktop browser game built to explore game development fundamentals and core design concepts. 

## Description  

A 2D side-scrolling action game developed with HTML Canvas, CSS, and JavaScript for game loops and event handling. 
Created with the help of tutorials by [Franks laboratory](https://www.youtube.com/playlist?list=PLYElE_rzEw_uryBrrzu2E626MY4zoXvx2) and expanded with custom mechanics.

## [Play Here](https://softwyredev.github.io/JS-Game/)

## Features   

- Sprite animations
- Collision detection
- Score tracking
- Particle effects
- Game loops and event handling

## Tech Stack  
HTML Canvas • CSS • JavaScript • GitHub Pages (Hosting)
